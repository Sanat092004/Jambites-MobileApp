# 🍔 Jambites - Traffic Jam Food Delivery App

Jambites is a native Android application that enables stuck motorists to order snacks, beverages, and essential medicines for delivery directly to their car window during traffic jams.

## 🛠️ Setup Instructions

### Prerequisites
- Android Studio Hedgehog or later
- JDK 17
- Android SDK 35
- Kotlin 1.9+

### Step 1: Clone and Open Project
```bash
git clone <repository-url>
cd Jambites
```
Open the project in Android Studio.

### Step 2: Configure API Keys

1. Copy the example properties file:
```bash
cp local.properties.example local.properties
```

2. Edit `local.properties` and add your API keys:
```properties
claude.api.key=sk-ant-xxxxx
maps.api.key=AIzaSyxxxxx
razorpay.key=rzp_test_xxxxx
```

### Step 3: Add Firebase Configuration

1. Go to [Firebase Console](https://console.firebase.google.com/)
2. Create a new project or use existing
3. Add Android app with package name: `com.jambites.app`
4. Download `google-services.json`
5. Place it in `app/` directory

### Step 4: Sync and Build

```bash
./gradlew clean build
```

### Step 5: Run the App

Connect a device or start an emulator, then:
```bash
./gradlew installDebug
```

## 📝 API Endpoints

The app expects the following backend endpoints:

- `GET /vendors/nearby` - Get nearby vendors
- `GET /vendors/{id}/menu` - Get vendor menu
- `POST /orders` - Place order
- `GET /orders/{id}/status` - Get order status
- `POST /orders/{id}/cancel` - Cancel order
- `GET /traffic/jam-check` - Check traffic jam status
- `POST /auth/otp/send` - Send OTP
- `POST /auth/otp/verify` - Verify OTP
- `GET /user/orders/history` - Get order history
- `POST /promo/validate` - Validate promo code
